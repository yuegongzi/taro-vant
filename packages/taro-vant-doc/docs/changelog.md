# 更新日志

### 介绍

Taro Vant 遵循 [Semver](https://semver.org/lang/zh-CN/) 语义化版本规范。

**发布节奏**

- 修订号：包含新特性和问题修复。
- 次版本号：包含新特性和较大的功能更新，向下兼容。
- 主版本号：包含不兼容更新和重大功能更新。

## 更新内容
